# bulksms-frontend

### Installation

Make sure you have [Node.js](https://nodejs.org/) v14+

- Clone the Repository

```sh
$ git clone git@gitlab.com:bulksms-smb/bulksms-frontend.git
```

- CD to the newly cloned Repository

```sh
$ cd bulksms-frontend
```

- Install dependencies

```sh
$ npm install
```

- Run the server

```sh
$ npm start
```
